## Общее описание
Плагин для подсветки отступов в коде разными цветами в зависимости от уровня вложенности.

## Функции

- `activate(context)` - вызывается при запуске расширения, получает обЪeкт context, используемый в дальнейшем для подписки на события

- `update()` - Функция обновления подсветки после произошедших изменений

- `deactivate()` - очистка ресурсов при отключении расширения
