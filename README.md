# rainbowindents README

# Коконов Никита Сергеевич М3100

## Общее описание
Подсветка отступов в коде

## Возможности

- Подсвечивает отступы разными цветами в зависимости от уровня вложенности
- Поддерживает до 5 уровней вложенности
- Автоматически обновляется при изменении кода
- Работает с отступами в 4 пробела

## Уровни подсветки

1. **Уровень 1** - светло-красный
2. **Уровень 2** - светло-голубой  
3. **Уровень 3** - светло-фиолетовый
4. **Уровень 4** - светло-зеленый
5. **Уровень 5** - светло-оранжевый
